# **App Name**: বাংলা ফন্ট সিডিএন

## Core Features:

- ফন্ট তালিকা: fonts.json থেকে ফন্টগুলি একটি প্রতিক্রিয়াশীল গ্রিড বিন্যাসে প্রদর্শন করুন। ফন্ট লিস্টিং এমন ভাবে ডিজাইন করা হবে যাতে সিম্পল এর মধ্যে গরজিয়াস হয়।
- ফন্ট অনুসন্ধান: অনুসন্ধান ইনপুট ক্ষেত্র ব্যবহার করে বাংলা নামে ফন্ট অনুসন্ধান করুন।
- ফন্ট আকার নিয়ন্ত্রণ: একটি স্লাইডার ব্যবহার করে পূর্বরূপ পাঠ্যের আকার সামঞ্জস্য করুন।
- ফন্ট কার্ড প্রদর্শন: নাম, ডিজাইনার, পূর্বরূপ পাঠ্য, বিভাগ, শৈলী, পছন্দ এবং ডাউনলোড সহ প্রতিটি ফন্টের বিবরণ একটি কার্ড বিন্যাসে প্রদর্শন করুন।
- ফন্ট ডাউনলোড: একটি ডাউনলোড বোতামের মাধ্যমে নির্বাচিত ফন্ট ফাইল ডাউনলোড করুন।
- লাইক কাউন্টার: ব্যবহারকারীদের ফন্ট পছন্দ করতে দিন, এবং লাইকের সংখ্যা সঞ্চয় করুন
- ডাউনলোড কাউন্টার: ফন্ট ডাউনলোডের সংখ্যা ট্র্যাক রাখুন।
- Preview: বিভিন্ন স্টাইলে ফন্টের প্রিভিউ দেখান
- লিংক কপি: <link> ও @import লিংক কপি করার ব্যবস্থা রাখুন

## Style Guidelines:

- প্রাথমিক রঙ: ডাউনলোড বোতামের জন্য লাল (#d32f2f), একটি স্পষ্ট কল টু অ্যাকশন তৈরি করে।
- পটভূমির রঙ: প্রধান পটভূমির জন্য হালকা ধূসর (#f5f5f5), একটি নিরপেক্ষ পটভূমি প্রদান করে।
- উচ্চারণ রঙ: ডাউনলোড বোতামের জন্য হোভারে গাঢ় লাল (#c62828), ব্যবহারকারীদের স্পষ্ট প্রতিক্রিয়া প্রদান করে।
- বডি এবং শিরোনাম ফন্ট: 'PT Sans', একটি মানবতাবাদী sans-serif যা শিরোনাম এবং বডি টেক্সট উভয়ের জন্য উপযুক্ত
- ওয়েবসাইটের ডিফল্ট ফন্ট এটা হবে @import url('https://cdnmx.pages.dev/assets/fonts/noto_serif_bengali/font.css');

- দৃষ্টি সহায়তার জন্য লাইক (হার্ট) এবং ডাউনলোড (ডাউনলোড) এর জন্য 'lucide-react' আইকন ব্যবহার করুন।
- ফন্ট তালিকার জন্য একটি প্রতিক্রিয়াশীল গ্রিড বিন্যাস প্রয়োগ করুন (মোবাইলে 1 কলাম, ট্যাবলেটে 2, ডেস্কটপে 3)।
- উন্নত ইন্টারঅ্যাক্টিভিটির জন্য ফন্ট কার্ড এবং বোতামগুলিতে সূক্ষ্ম রূপান্তর এবং হোভার প্রভাব ব্যবহার করুন।
